About AMG's iRacing Planner
Hey there!
My name is Albert and I am an enthusiast of simRacing. Some years ago I registered into iRacing and with the iRacing's PDF schedule I think it is quite complicated to plan a season depending on your preferences, cars and tracks... so after some seasons I decided to create this simple tool to help simRacers planning their season.
Instructions:
Just select your owned cars and tracks in the corresponding tabs and you will see your season in the dashboard tab.
Some notes:
  • At this moment, this tool uses a json produced with an iRacing API call and for now this needs to be updated manually every season. Please go to my Github repository to download a release for the current season. I will be updating it until I found an automatic solution for this.
  • The list of cars and tracks are automatically generated with the mentioned json. It is possible that if a car or a track is not used and does not appear in the json, you won't find it in the list.
  • In the mentioned json it is not specified which cars and tracks are free, so at this moment please select all cars and tracks that you own.
  • Series with less than 12 races on the season are not shown properly, if you want to race this series please be careful because the weeks in this cases are incorrect
