About AMG's iRacing Planner
Hey there!
My name is Albert and I am an enthusiast of simRacing. Some years ago I registered into iRacing and with the iRacing's PDF schedule I think it is quite complicated to plan a season depending on your preferences, cars and tracks... so after some seasons I decided to create this simple tool to help simRacers planning their season.
Instructions:
When the tool starts, just select your owned cars and tracks in the corresponding tabs and you will see your season in the dashboard tab.
Some notes:
  • This app uses json files produced with iRacing API calls for obtaining the current season, cars and tracks. These json files need to be updated manually each season. For updating them, please follow the steps mentioned in the section 'How the data is obtained' from here.
  • You can download latest releases in here.
  • The list of cars and tracks are automatically generated with the mentioned json. It is possible that if a car or a track is not used and does not appear in the json, you won't find it in the list.
  • Series with less than 12 races on the season are not shown, as this was confusing because the time of the races does not appear in the json file. If you want to race this series please take into account that this series does not appear.
